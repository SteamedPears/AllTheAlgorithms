Algorithms Notes

By Alexis Beingessner, Simon Pratt

Download Download fork Algorithms Notes on GitHub PDF

These notes are intended for study of the material for COMP3804 at Carleton University, and should be suitable as the notes for a first course in algorithms.

Assumed knowledge includes: big-Oh, big-Omega, big-Theta.

Topics include:

  • Fibonacci numbers
  • integer multiplication
  • recurrences
  • heaps
  • selection
  • union-find
  • graphs
  • depth first search
  • minimum spanning tree.
  • single source shortest path (Dijkstra's algorithm)
  • complexity classes

Intended topics also include:

  • dynamic programming
  • NP-completeness
